# Text Editor

The Text Editor component can receive the following props:

<Props of={EditText} />

It's necessary to specify the prop `skinUrl` of component to point the location of the editor stylesheet.
Skins can be found on folder `node-modules/tinymce/skins/ui` and copied to public asset folder of your application.
You can check our <Link href="https://github.com/tecsinapse/text-editor/tree/master/storybook-public/oxide">github</Link> to view how the folder is mounted according to code snippet below.
